# LinkedIn Clone

This project is a simplified clone of Social medias (mainly Linkdin and twitter(X))

**Features:**

* User registration and login
* User profile and details(study, work)
* Post creation, edit, update and delete
* Other Post like, comment and repost, comment like
* Follow unfollow other users,see their following, followers lists 


**Live Demo:** https://linkdin-clone-nine.vercel.app/

* [List of technologies used: React, firebase, tailwind]
